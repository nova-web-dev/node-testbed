# Carl's Node Testbed

This is a simple node repo with many different scripts that I have written
over time.

### Setup
- Clone the repo
- Run gulp scripts
- Run with node index

### Commands:

    $ git clone URL_TO_COME_HERE
    $ cd node-testbed
    $ npm install -g gulp
    $ gulp
